Windows 上使用 Shadowsocks 教程

July 6, 2017

Windows

本文简要叙述 Windows 下（适用于 Windows XP，7，8，10）Shadowsocks 客户端的安装、配置 和使用过程。

下载软件

下载 Windows 下的 Shadowsocks 软件，点击下载 shadowsocks-2.5.8.zip 。

解压运行

下载后解压，解压后运行 Shadowsocks 。

图 1. Shadowsocks 图标

配置帐号

首次运行，会弹出编辑服务器窗口，根据提供的帐号信息，正确填写服务器地址、端口、密码 和加密方式，然后点确定。

图 2. 配置界面

点确定后，右下角弹出提示。

图 3. 任务栏图标

右键程序图标，弹出主菜单，勾选 “启用系统代理”。

图 4. 启用 Shadowsocks

验证是否成功

打开任意浏览器（IE / Chrome / Firefox 等），即可开始科学上网。

试试被墙网站：Google，Twitter，YouTube

图 5. 打开 Twitter 试试

Tips

  1. 如果软件无法运行，弹出需要安装 .net ，请前往微软官网安装对应系统的 .net；
  2. 加密方式（算法）默认为 aes-256-cfb；
  3. 验证时，先用 IE 打开测试，出问题的可能小，其他浏览器注意代理插件的相关设置；
  4. 首次运行后，点击 “从 GFWlist 更新本地 PAC 文件” ，然后设置代理模式为 “PAC 模式”， 这样对被墙网站会自动启用代理；
  5. 如果无法连接，试试切换到 “全局代理” 模式；
  6. 如果稳定性欠佳，尝试切换不同节点。
